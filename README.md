Advanced Gomoku (Caro Game)
A high-performance web-based Gomoku game powered by Python, JavaScript, and MySQL. This project features an intelligent AI opponent with multiple difficulty levels and a real-time online multiplayer mode.

🚀 Key Features
Single Player (AI Mode): Challenge the computer with 3 difficulty levels:

Easy: Simple random or basic defensive moves.

Medium: Balanced offensive and defensive tactics.

Hard: Advanced Minimax algorithm for strategic play.

Persistent Data: Integrated with MySQL to store user accounts, match history, and global rankings.

Modern Interaction: Seamless gameplay with JavaScript event handling and CSS animations.

Backend Logic: Robust game rules and database management handled by Python.

🛠️ Tech Stack
Backend: Python Flask, Flask-SocketIO.

Database: MySQL WorkBench (User data, History, Leaderboard).

Frontend: HTML5, CSS, JavaScript (Client-side logic & Clicking).

AI Algorithm: Heuristic evaluation or Minimax for the bot.

Authentication: Werkzeug Security

Installation & Setup

1. Clone the project:

git clone https://github.com/SyPhu243/DOANCARO

2. Database Configuration:

Import the provided SQL schema into your MySQL server.

Update your database credentials in the Python configuration file.

3. Install Requirements:

pip install -r requirements.txt

4. Launch the Game:

python app.py
